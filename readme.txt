Embed a Monero Miner in your Website
== Description ==

Go to a thingspeak channel like https://thingspeak.com/channels/123778/
Take the Channel ID and put a widget with the id  on your site

An example id is set by default

== Installation ==
1. Register at https://coin-hive.com/
2. Create a Page
3. Upload the plugin to your blog, Activate it.

Wordress:

4. Go to Settings -> Monero and add your  Site Puplic Key
